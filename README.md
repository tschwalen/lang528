## Installation

#### Dependencies:

- cmake, make, and a C++17 compiler
- [Niels Lohmann's JSON library for C++](https://github.com/nlohmann/json)
  - download version 3.11.2 as a header only library and place it in the include directory like so:
  ```
  include
   | ...
   \- nlohmann
       \- json.hpp
  ```
